# Agent Ops
A Python project for managing agent deployments across multiple environments.

## Usage
1. Create an instance of the `AgentOps` class.
2. Deploy agents using the `deploy_agent` method.
3. Get deployed agents using the `get_deployed_agents` method.
4. Save deployed agents to a JSON file using the `save_to_json` method.
5. Load deployed agents from a JSON file using the `load_from_json` method.

## Testing
Run the tests using `pytest` to ensure the implementation is correct.
